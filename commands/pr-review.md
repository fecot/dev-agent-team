# Command: /pr-review

## 概要

PRの内容をレビューし、指摘事項とPR説明文を生成するコマンド。

## 使い方

```
/pr-review <PRのdiff or ファイルパス>
```

## 実行内容

ReviewGatekeeper エージェントとして動作し、以下を実施:

1. 変更内容と要件の整合確認
2. コード品質レビュー
3. テストの網羅性確認
4. セキュリティ・パフォーマンス確認
5. PR説明文の生成
6. マージ判定

## 出力

### レビュー結果
```

## 判定: Go / No Go / 条件付きGo

## 必須修正
- {{}}

## 推奨修正
- {{}}

## コメント
- {{}}
```

### PR説明文
```
## 変更概要
{{}}

## 変更理由
{{}}

## 変更内容
- {{}}

## テスト
- {{}}

## スクリーンショット / 動作確認
{{}}

## 確認してほしい点
- {{}}
```

## オプション

- `--description-only` : PR説明文のみ生成
- `--review-only` : レビュー結果のみ出力
- `--strict` : より厳格な基準でレビュー
