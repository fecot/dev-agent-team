# Troubleshooting

## チャットアプリやログ出力で `[CLAUDE.md](http://CLAUDE.md)` のような表記が見える

一部のチャットアプリや入力欄では、`*.md` のような文字列がペースト時に自動的にマークダウンリンクに変換されることがあります。これは表示・入力時の変換であり、実ファイルの内容には影響しません。

### 実ファイルの内容を確認する方法

表示変換の影響を受けず、実ファイルのバイト列を確認するには `xxd` や `od -c` を使ってください:

```sh
xxd <ファイルパス> | head
od -c <ファイルパス> | head
```

これらは表示変換の影響を受けないので、ファイルの実際の内容を確認できます。

### 回避策

多くのチャットアプリでは、コードブロック(```) で囲ってから貼り付けると自動変換が抑制されます。ファイル内容を正確に共有したい場合に有効です。
