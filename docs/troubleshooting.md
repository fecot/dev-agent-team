# Troubleshooting

dev-agent-team を使う上で遭遇しがちな表示・動作の癖をまとめます。仕様の不具合ではなく、**ターミナル / ターミナルアプリ側の表示変換** に起因することが多いので、原因の切り分けに役立ててください。

---

## チャットに貼った時に CLAUDE.md のような表記になる

### 症状

チャット欄やドキュメントに `*.md` のような文字列を貼ると、勝手に `**.md**`（マークダウン強調）や、ファイル名がリンク化されて表示されることがあります。

### 原因

cmux など一部のターミナル / ターミナルアプリは、**`*.md` のような文字列を自動でマークダウンリンク・強調記法に変換** します。これは **表示時の変換** であり、**実ファイルの内容には影響しません**。

### 確認方法

実ファイルの内容を確認したい場合は、`xxd` や `od -c` を使ってください:

```sh
xxd <ファイルパス> | head
od -c <ファイルパス> | head
```

これらは生バイト列をそのまま表示するため、ターミナルの表示変換が混入しません。

### よくあるケース

- `/adopt-project` の `--dry` 出力で `CLAUDE.md` が `**CLAUDE.md**` のように見える
- `.gitignore` の例の `*.bak.*` 行が `\*.bak.\*` のようにバックスラッシュ付きで貼り付けされる

これらはすべて **表示の問題** で、実ファイル / 実コマンドの動作には影響しません。
